# ICS 4111: Embedded Systems & IoT
## Practical Exercise 3: Embedded System Design

This repository contains the documentation and findings for Practical Exercise 3 (April - July 2026 semester), focusing on the practical demonstration of Ohm's Law and Pulse Width Modulation (PWM).

### Repository Structure
* **Branch:** `Embedded-Design`
* **Main Report:** `ICS_4111_Practical_Exercise_3_Group_X.pdf` *(Contains Experiments 1–4, data analysis, circuit photographs, and reflections)*

### Core Experiments Covered
1. **Experiment 1:** Resistor color-code decoding vs. digital multimeter verification.
2. **Experiment 2:** Microcontroller LED blink deployment and PWM light intensity control via a potentiometer.
3. **Experiment 3:** Series circuit implementation, voltage drop measurements, and Ohm's Law current estimation.
4. **Experiment 4:** Parallel circuit implementation, branch current calculations, and circuit behavior analysis.
